A collection of React projects demonstrating fundamental concepts and best practices. This repository contains hands-on examples covering JSX, components, props, state management, event handling, and styling.

## 🎯 Learning Objectives

### 01_demo - React Fundamentals
- **JSX Syntax**: Understanding JSX expressions and embedding JavaScript
- **Component Creation**: Building reusable functional components
- **Props**: Passing and using props for component communication
- **Conditional Rendering**: Displaying content based on conditions
- **List Rendering**: Mapping arrays to JSX elements
- **Dynamic Content**: Displaying dates, user data, and calculated values

### 02_demo - Advanced Concepts
- **State Management**: Using `useState` hook for component state
- **Event Handling**: Click, copy, and mouse events
- **Styling**: Inline styles, CSS modules, and external stylesheets
- **Array State**: Adding, removing, and updating array elements
- **Object State**: Modifying object properties in state
- **React Icons**: Integration of icon libraries

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/react-learning-journey.git
cd react-learning-journey
```

2. Choose a project and install dependencies:
```bash
# For project 1
cd 01_demo
npm install

# For project 2
cd 02_demo
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`
